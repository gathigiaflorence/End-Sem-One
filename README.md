# End-Sem-One
A simple API-driven asynchronous web app that lets users search for any city in the world and see the current weather, including temperature, wind speed, and weather icon. Built with HTML5, CSS3, and JavaScript ES6+.

Features
Search any city worldwide and get the latest weather.
Dynamic weather icons based on weather conditions (sunny, cloudy, rainy, etc.).
Loading indicator while fetching data.
Handles errors for:
Empty input
City not found
Network or API failures
Responsive design with Flexbox.
Accessible: alt text for images and labels for input.
